# Port Status Checker

## Description
A simple Python program that checks whether a port is open or closed on a given host.

## Features

- Checks port status
- Accepts host and port as input
- Displays Open or Closed

## Run

```bash
python Port_Status_Checker.py
```

## Sample Output

```text
Enter Host/IP Address: google.com
Enter Port Number: 80

Port 80 is OPEN
```
